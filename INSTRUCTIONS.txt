STEP 8: MAKE FORMS WORK (IMPORTANT!)
================================================================================

Right now, the forms will display but won't actually SEND emails.
You need to connect them to a form processor.

OPTION A: Use Formspree (FREE - RECOMMENDED):
---------------------------------------------
1. Go to www.formspree.io
2. Sign up for free account
3. Create a new form
4. Copy the form endpoint URL they give you
5. In File Manager, edit index.html, training.html, and contact.html
6. Find: <form>
7. Change to: <form action="YOUR_FORMSPREE_URL" method="POST">
8. Save file
9. Test form - emails will now come to info@usewiserly.com

OPTION B: Use GoDaddy Email Hosting:
------------------------------------
1. Contact GoDaddy support
2. Ask them to help connect forms to your email
3. They may charge a small fee for this service

TROUBLESHOOTING
================================================================================

PROBLEM: Files uploaded but website doesn't show
SOLUTION: Make sure files are in "public_html" folder, not a subfolder

PROBLEM: Images don't display
SOLUTION: Check that image filenames match EXACTLY (case-sensitive):
         - square_profile_i.png (lowercase)
         - wide_bannetter_F.png (capital F)

PROBLEM: Styling looks wrong
SOLUTION: Make sure styles.css uploaded correctly
         Make sure it's in same folder as HTML files

PROBLEM: Website shows old content
SOLUTION: Clear your browser cache (Ctrl+F5 or Cmd+R)
         Try viewing in "Incognito/Private" mode

PROBLEM: GoDaddy won't let me upload files
SOLUTION: You may need to upgrade to a hosting plan that allows HTML uploads
         Contact GoDaddy support: 1-480-505-8877
